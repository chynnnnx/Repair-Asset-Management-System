# Repair-Asset-Management-System
## Overview  
This project is a **Repair Asset Management System** built with **Blazor WebApp**, **ASP.NET Core**, and **MudBlazor** for the UI.  
It helps manage assets, track repair requests, and monitor their status in one place with a clean, modern interface.  

## Prerequisites  
Before running this project, make sure you have:  
- .NET 9 SDK  
- Visual Studio 2022 with **ASP.NET and web development workload**  
- Microsoft SQL Server 2022  
- (Optional) SQL Server Management Studio (SSMS) for database management  


## Features 
- Add, update, and delete assets  
- Create and manage repair requests  
- Track asset repair history  
- Assign technicians to tasks  
- Dashboard for quick insights  
- Responsive UI with MudBlazor components

  ## Tech Stack  
- **Frontend**: Blazor WebApp, MudBlazor, CSS  
- **Backend**: ASP.NET Core (C#)  
- **Database**: Microsoft SQL Server (MSSQL)
